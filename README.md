Prerequisites
-Node.js versi 16 atau lebih baru
-npm
-Git
Setup
-Clone repository ini ke komputer lokal
-Masuk ke folder project
-Jalankan npm install untuk menginstall semua dependency
Run Test
Jalankan npx cypress open untuk menjalankan test melalui UI
Jalankan npx cypress run untuk menjalankan test melalui terminal
